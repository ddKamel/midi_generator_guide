# midi_generator_guide
A students project at JLU Gießen where I create a MIDI Generator for the module Deep Learning

# Main notebook:

The main notebook with a detailed explanation of how everything works is the file kamel_midi_project.ipynb

[CLICK HERE TO OPEN THE HTML](./kamel_midi_project.html)

# Included

No models are included due to their filesize. Please train them yourself.
Some samples generated by the LAKH dataset are included to listen to the outstanding musical performance. (Some are bugged)
# Usage

Simply clone this project, hit pip install and then the notebooks should be runable. Keep in mind that you have to download the datasets yourself. They are linked in the main notebook.

Please adjust your paths accordingly if you use a different structure.
In some notebooks you need to adjust which architecture you want to use and which datasets.
Save your datasets in the folder datasets/midi. You can however use whatever suits you best.
